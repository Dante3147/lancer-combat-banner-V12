# O que é isso? 

Lancer Combat Banner é um módulo que adiciona uma exibição visual sempre que a rodada muda e sempre que uma unidade é ativada.. 

[combatbanner.webm](https://github.com/hiddenkrypt/LancerCombatBanner/assets/5739024/6a12e145-d142-4f04-9f4c-d3ab91a636be)

Vídeo do autor original

# Features

A cor muda automaticamente de acordo com a cor do jogador e do mestre. Isso pode ser alterado para uma cor definida nas configurações para todos os banners.

Também gera uma mensagem no chat quando um personagem é ativado no rastreador de combate e quando uma nova rodada começa. Isso pode ser desativado nas configurações..
